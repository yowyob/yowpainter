\section{Pipeline de build et déploiement}

\begin{enumerate}[noitemsep]
  \item \textbf{Build} : \texttt{mvn clean install} (compilation, tests unitaires)
  \item \textbf{Tests} : \texttt{mvn test} (H2 en mémoire pour tests)
  \item \textbf{Package} : JAR exécutable Spring Boot (\texttt{target/yowpainter-backend-*.jar})
  \item \textbf{Migrations} : Flyway exécuté au démarrage (schéma public + tenants actifs)
  \item \textbf{Démarrage} : \texttt{java -jar app.jar} ou \texttt{.\textbackslash scripts\textbackslash run-local.ps1}
  \item \textbf{Vérification} : \texttt{GET /api/public/health} + Swagger UI
\end{enumerate}
