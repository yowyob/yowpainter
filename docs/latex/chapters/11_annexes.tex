\section{Checklist de mise en production}

\begin{enumerate}
  \item Créer \texttt{ClientApplication} \texttt{yowpainter-backend} dans le Kernel
  \item Configurer toutes les variables d'environnement (pas de secrets en dur)
  \item Définir \texttt{KSM\_KERNEL\_AUTO\_PROVISION\_ARTISTS=false} pour validation manuelle
  \item Configurer credentials bootstrap admin + MFA
  \item Vérifier connexion PostgreSQL et droits création schémas
  \item Renseigner \texttt{BACKEND\_URL} (callback URL publique joignable par le Kernel)
  \item Configurer SMTP pour e-mails transactionnels
  \item Activer Kafka si synchronisation commandes requise
  \item Sécuriser routes chat (\texttt{permitAll} → authentification requise)
  \item Tester health checks et Swagger
  \item Exécuter \texttt{mvn test} en CI
\end{enumerate}
